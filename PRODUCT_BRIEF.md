# Product Brief: ConstructGraph OS

## Mission
ConstructGraph OS is an advanced AI construction project-control platform. It connects documents, drawings, contracts, RFIs, submittals, schedules, emails, costs, procurement, and site evidence into one live intelligence system.

## Tagline
Autonomous project-control intelligence for construction teams.

## Core Differentiator
The **Project Truth Graph**. Every project entity is a node, and every relationship is an edge. It does not just retrieve text; it connects facts to present a live picture of project risk, claims, and readiness.

## Target Users
1. Project Managers
2. Contract Administrators / Quantity Surveyors
3. Site Engineers / Superintendents
4. Design Managers
5. Procurement Managers
6. Owners / Developers
7. Claims Consultants / Delay Analysts

## Core Screens & Features
- **Project Command Center**: Executive-level dashboard showing project health, risks, exposure, and blockers.
- **Risk Radar**: Ranked risk dashboard with severity, exposure, and recommended actions.
- **Project Truth Graph Explorer**: Graph UI showing relationships between entities.
- **Contract Obligation Register**: Auto-generated register tracking obligations and deadlines.
- **Claims / Variation Lab**: Detects possible claims/variations, identifies missing evidence, and drafts notices.
- **Drawing & Spec Contradiction Report**: Identifies conflicts across the document set.
- **Schedule Risk Dashboard**: Tracks blocked activities and delay probabilities.
- **Procurement Risk Dashboard**: Tracks long-lead items against site need dates.
- **Evidence Locker**: Auto-organizes evidence for risks and claims.
- **Handover Readiness**: Tracks essential closeout documentation and defect resolution.

## MVP Scope
The initial MVP will operate on a synthetic demo project: **Riverside Medical Center Expansion — Phase 2**.
It will utilize deterministic logic (rule-based scoring and extraction) to demonstrate the product's value proposition before moving to full AI/LLM integrations.
